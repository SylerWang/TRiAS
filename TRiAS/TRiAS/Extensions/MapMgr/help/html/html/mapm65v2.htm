Derzeit werden von TRiAS® die folgenden Rasterdatenformate unterstützt: 

Name      Farbtiefe   Kompression
TIFF      1 BpP       G3, G4 (nur lesen)
Tiff 1 Bit pro Pixel, unkomprimiert, G3 und G4-komprimiert
Tiff 4 und 8 Bit pro Pixel, unkomprimiert, RLE-komprimiert
Windows-Bitmap 1,4,8 Bit pro Pixel, unkomprimiert, RLE-komprimiert
