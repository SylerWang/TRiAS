TRiAS GmbH Potsdam, InterTRiAS V3.12, Potsdam, den 6. Juni 2000
Letzte Informationen

In dieser Datei sind folgende Punkte enthalten:
	1. Systemvorraussetzungen
	2. Installation
	3. Handbuch
	4. Auftretende Fehler


1. Systemvorraussetzungen für InterTRiAS
- WindowsNT V4.0 (SP3) oder Windows 95/98 oder neuer
- Internet-Explorer V4.0 (SP1) oder neuer. Sie können dieses Programm direkt 
von der vorliegenden CD installieren. Bitte starten Sie dazu im Verzeichnis 
\IE401 das Programm ie4setup.exe.


2. Installation von InterTRiAS
- Zur Installation starten Sie bitte die Datei InterTRiAS_Analyse.exe im 
Verzeichnis \InterTRiAS der CD. 
- Unter WindowsNT muá InterTRiAS z.Zt. von einem Nutzer installiert werden, 
welcher Administratorrechte besitzt. Die Installation wird anderenfalls 
abgebrochen.
	

3. Handbuch
- Das komplette Handbuch ist im Verzeichnis \Handbuch dieser CD enthalten.
Es liegt im PDF-Format vor. Sollten Sie keine Möglichkeit haben, dieses Format
anzuzeigen, so sollten Sie das Program 'Acrobat Reader' installieren, dessen
Installationsprogramm sich im Verzeichnis \AcroRead befindet.


4. Auftretende Fehler
Wir möchten Sie bitten bei auftretenden Fehlern im Programm oder Handbuch bzw. 
Unstimmigkeiten zwischen beiden an uns ein Fehlerprotokoll zu senden.
